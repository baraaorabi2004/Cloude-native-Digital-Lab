المختبر الرقمي السحابي المستدام لطلاب جامعة البلقاء التطبيقية (BAU-FET)
🚀 رؤية المشروع (The Vision)
تحويل الموارد والأجهزة في جامعة البلقاء التطبيقية إلى "مختبر رقمي سحابي" يوفر بيئة تدريبية ذكية للأمن السيبراني والشبكات. يركز المشروع على الاستدامة عبر تقليل النفايات الإلكترونية (Zero E-waste Goal) وإطالة عمر الأجهزة التعليمية من خلال نقل معالجة البيانات الثقيلة إلى السحابة.

🛠️ الترسانة التقنية (Tech Stack)
1. البنية التحتية والافتراضية (Virtualization)
Docker & Google Cloud Run: لتغليف النظام واستضافته بهيكلية "خالية من السيرفرات" (Serverless).

KasmVNC & noVNC: توفير سطح مكتب رسومي فائق السرعة يعمل بالكامل داخل المتصفح (WebGL/WebSockets).

Xvfb & XFCE4: لإنشاء شاشة وهمية وبيئة سطح مكتب خفيفة الوزن تضمن أداءً مستقراً بأقل الموارد.

2. محرك الشبكات والذكاء الاصطناعي
Cisco Packet Tracer 8.x: المحرك الأساسي لمحاكاة وتصميم الشبكات.

Gemini 1.5 Flash (AI Robbie): العقل المدبر للمساعد الذكي الذي يحلل أخطاء الطلاب ويقدم دعماً تقنياً فورياً.

Python & Flask: لربط واجهة المستخدم بمحرك الذكاء الاصطناعي عبر APIs.

🤖 الميزات الذكية والاستدامة (Smart Features & Sustainability)
AI Lab Tutor (روبي): مساعد تقني متخصص في الأجهزة والشبكات، يساعد في تصحيح أوامر CLI وشرح بروتوكولات OSPF وVLANs.

إطالة عمر الأجهزة (Hardware Lifecycle Extension): تقليل البصمة الكربونية للجامعة عبر السماح للأجهزة القديمة بتشغيل أحدث برامج المحاكاة عبر المتصفح.

إدارة الموارد التنبؤية: توزيع الأحمال بذكاء لتقليل استهلاك الطاقة في مراكز البيانات.

🐳 هيكلية الحاويات (Docker & Deployment)
يعتمد بناء المختبر على ملف Dockerfile احترافي يدير الطبقات التالية:

قاعدة النظام: Ubuntu 22.04 مع إصلاح كافة التبعات (Dependencies) مثل libqt5networkauth5.

التخصيص البصري: واجهة مستخدم بألوان "الأزرق السماوي" لتناسب الهوية البصرية للمشروع.

نص التشغيل (Startup Script): ملف startup.sh يضمن تشغيل (VNC، المختبر، وRobbie) في آن واحد بتناغم تام.

أوامر التشغيل السريع (Deployment Commands)
بناء النظام:

Bash
gcloud builds submit --tag gcr.io/[PROJECT_ID]/cisco-lab .
النشر على السحابة:

Bash
gcloud run deploy cisco-real-lab --image gcr.io/[PROJECT_ID]/cisco-lab --memory 4Gi --cpu 2 --al
```
