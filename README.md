# 🌍 Cloud-Native Digital lab 🛡️
**مشروع مختبر رقمي سحابي مستدام لطلاب جامعة البلقاء التطبيقية (BAU-FET)**

## 🚀 رؤية المشروع (The Vision)
تحويل الموارد والأجهزة القديمة في جامعة البلقاء إلى "مختبر رقمي سحابي" يوفر بيئة تدريبية ذكية للأمن السيبراني، مع التركيز على الاستدامة وتقليل النفايات الإلكترونية.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Cloud Infrastructure:** Docker & K3s (Kubernetes).
- **AI Intelligence:** Google Gemini 1.5 Flash (AI Lab Tutor).
- **Web Access:** KasmVNC for browser-based lab sessions.
- **Frontend:** Tailwind CSS Dashboard.

## ✨ الميزات الذكية (AI Features)
- **AI Lab Tutor:** مساعد ذكي يحلل أخطاء الطالب في Kali Linux ويقدم تلميحات تعليمية فورية باستخدام Gemini API.
- **Predictive Resource Management:** توفير الطاقة عبر توزيع الأحمال بذكاء بناءً على أوقات الذروة.

## ♻️ الاستدامة والأثر (Sustainability & Impact)
- **Hardware Lifecycle Extension:** إطالة عمر أجهزة الجامعة بنسبة تصل لـ 5 سنوات.
- **Zero E-waste Goal:** تقليل البصمة الكربونية والنفايات الإلكترونية في الأردن.
